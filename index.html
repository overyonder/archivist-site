    NAS Historian

    A storage-side Rust service watches the Vault and Intake directories, hashes changed paths beside the data, and publishes immutable MessagePack snapshot generations.

    An authenticated WebSocket streams status and generation changes to Archivist. The desktop updates while connected without starting another full hash pass, and retains the last local snapshot for offline browsing.

    ZFS snapshot comparison, statx identity reuse, fs-verity digests, and digest xattrs are planned evidence providers; the current implementation uses recursive filesystem notifications and SHA-256.
